## 🎮 核心系统分析

### 1. 游戏主循环 (main.py)

- **职责**: 游戏初始化和主循环
- **关键组件**: Dashboard, Level, Menu, Sound
- **流程**: 菜单 → 游戏 → 重启

### 2. 关卡系统 (Level.py)

- **职责**: 关卡加载、渲染、实体管理
- **数据格式**: JSON格式的关卡文件
- **核心功能**:
  - `loadLevel()` - 加载关卡数据
  - `loadEntities()` - 加载实体
  - `loadLayers()` - 加载背景层
  - `loadObjects()` - 加载静态对象

### 3. 实体系统 (entities/)

- **基类**: EntityBase - 所有实体的基础类
- **马里奥**: Mario.py - 玩家角色，包含状态管理
- **敌人**: Goomba.py, Koopa.py - AI敌人
- **道具**: Coin.py, Mushroom.py - 可收集道具

### 4. 精灵图系统 (sprites/)

- **配置格式**: JSON格式定义精灵图位置和属性
- **动画系统**: 基于精灵图序列的动画
- **缩放支持**: 支持不同尺寸的精灵图

## 📊 关卡数据结构分析

### JSON关卡文件结构

```json
{
    "id": 0,                    // 关卡ID
    "length": 60,               // 关卡长度
    "level": {
        "objects": {            // 静态对象
            "bush": [[x,y], ...],     // 灌木
            "cloud": [[x,y], ...],    // 云朵
            "pipe": [[x,y,height], ...], // 管道
            "ground": [[x,y], ...]     // 地面
        },
        "layers": {             // 背景层
            "sky": {"x": [0,60], "y": [0,13]},
            "ground": {"x": [0,60], "y": [14,16]}
        },
        "entities": {           // 动态实体
            "CoinBox": [[x,y], ...],    // 金币箱
            "Goomba": [[x,y], ...],     // 敌人
            "coin": [[x,y], ...],       // 金币
            "RandomBox": [[x,y,item], ...] // 随机道具箱
        }
    }
}
```

## 🛠️ 增加关卡的方案

### 方案1: 手动编辑JSON文件 (推荐)

**优点**:

- 简单直接，无需额外工具
- 完全控制关卡设计
- 易于版本控制

**步骤**:

1. 复制现有关卡文件 (如 Level1-1.json)
2. 修改关卡ID和长度
3. 调整对象和实体位置
4. 在代码中添加新关卡引用

### 方案2: 开发关卡编辑器

**优点**:

- 可视化编辑
- 实时预览
- 提高效率

**实现思路**:

```python
# 简单的关卡编辑器示例
class LevelEditor:
    def __init__(self):
        self.tiles = []
        self.entities = []
        self.current_tool = "ground"
    
    def add_tile(self, x, y, tile_type):
        self.tiles.append([x, y, tile_type])
    
    def save_level(self, filename):
        level_data = {
            "id": self.level_id,
            "length": self.level_length,
            "level": {
                "objects": self.tiles,
                "entities": self.entities
            }
        }
        with open(f"levels/{filename}.json", "w") as f:
            json.dump(level_data, f)
```
